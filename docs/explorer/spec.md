# explorer 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ファイル一覧表示 | タブに指定フォルダの内容（名前・更新日時・種類・サイズ）をテーブル表示する |
| 2 | フォルダナビゲーション | ダブルクリックでフォルダに移動、「上へ」「戻る」「進む」で履歴移動する |
| 3 | タブ管理 | ペイン内でタブの追加・切替・クローズができる。「＋」ボタンでアクティブタブを複製する |
| 4 | 画面分割（ペイン管理） | ペインを縦・横に分割/削除でき、スプリッターでサイズ調整できる |
| 5 | アドレスバー | 各ペイン上部にアクティブタブのフォルダパスを表示し、パス入力で直接移動できる |
| 6 | セッション保存・復元 | 終了時にペイン構成・タブ・パス・アクティブ状態を保存し、次回起動時に復元する |

## 画面・操作フロー

1. アプリを起動する → 前回のセッションがあれば復元、なければホームフォルダを開いた 1 ペイン 1 タブで表示
2. ファイル一覧のフォルダをダブルクリックする → そのタブがフォルダ内に移動し、アドレスバーが更新される
3. タブバーの「＋」ボタンを押す → アクティブタブと同じパスの新しいタブが右隣に開き、アクティブになる
4. メニューまたはショートカットで「分割」を実行する → アクティブペインが分割され、新ペインに同じパスのタブが開く
5. アドレスバーにパスを入力して Enter → アクティブタブがそのパスに移動する
6. アプリを閉じる → 現在の状態が自動保存される。再起動すると手順 1 で復元される

## 各画面 / 状態の詳細

### メインウィンドウ

- **表示内容**: 1 個以上のペインを分割レイアウトで表示。各ペインは「タブバー」「アドレスバー」「ファイル一覧」で構成
- **操作**: ペインのクリック（アクティブペイン切替）、スプリッターのドラッグ（サイズ調整）、分割の追加/削除
- **遷移先**: なし（単一ウィンドウ）

### ペイン

- **表示内容**: タブバー（開いているタブ＋「＋」ボタン）、アドレスバー、アクティブタブのファイル一覧
- **操作**:
  - タブクリック → アクティブタブ切替
  - タブの「×」→ タブクローズ（最後の 1 タブを閉じたらペインを閉じる。最後のペインの場合はホームのタブを開き直す）
  - 「＋」→ アクティブタブの複製
- **遷移先**: なし

### ファイル一覧

- **表示内容**: フォルダ→ファイルの順で、名前・更新日時・種類・サイズを表示。列ヘッダクリックでソート
- **操作**: シングルクリック（行選択）、フォルダのダブルクリック（移動）、ツールバーの「上へ」「戻る」「進む」
- **遷移先**: 同一タブ内でのパス変更

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 存在しないパスを入力 | アドレスバー下に「パスが見つかりません」を表示し、移動しない |
| アクセス権のないフォルダ | 一覧領域に「アクセスが拒否されました」を表示。タブは維持 |
| セッション内のパスが復元時に存在しない | そのタブはホームフォルダで開き、通知を表示 |
| セッションファイルが破損 | 既定状態（1 ペイン 1 タブ、ホーム）で起動 |
