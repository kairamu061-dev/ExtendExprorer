# shortcuts 概要

## 目的・背景

エクスプローラーと同じキーボード・マウス操作を提供する。
ユーザ要望（2026-07-19）「コピペ等のショートカットを使えるようにしたい」「マウスのサイドボタンの進む・戻るに対応したい」を担う。

## スコープ

### 作るもの

- ファイル一覧のキーボードショートカット: Ctrl+C（コピー）/ Ctrl+X（切り取り）/ Ctrl+V（貼り付け）/ Delete（ごみ箱へ削除）
- マウスサイドボタン: XButton1=戻る / XButton2=進む（ペイン単位、アクティブタブの履歴を移動）
- クリップボード形式はエクスプローラー互換（CF_HDROP＋Preferred DropEffect）。エクスプローラーとの相互コピペが可能

### 作らないもの

- F2 リネーム・Ctrl+A 全選択以外の追加キー（将来対応。Ctrl+A は ListView 標準で動く場合あり）
- ツリー・タブバー上でのショートカット

## 制約

- 実ファイル操作は IFileOperation（BUG-005 基盤）に委譲。確認・進捗・衝突ダイアログはシェル任せ
- AOT 互換（LibraryImport / KeyboardAccelerator のみ）

## 完了条件

- 一覧で Ctrl+C→Ctrl+V の複製、Ctrl+X→Ctrl+V の移動、Delete のごみ箱削除ができる
- エクスプローラーでコピーしたファイルを本アプリで貼り付け（およびその逆）ができる
- サイドボタンで戻る/進むが動く（履歴端では無反応）
