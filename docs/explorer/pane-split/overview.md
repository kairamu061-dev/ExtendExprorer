# pane-split 概要

## 目的・背景

複数フォルダを同時に見比べながら作業するための画面分割機能。
ユーザ要求「画面分割できる」を担う。参考ツール（Tablacus Explorer）の 2×2 分割が目標形。

## スコープ

### 作るもの

- アクティブペインの縦分割・横分割（新ペインには同じパスのタブを 1 つ開く）
- ペインのクローズ（分割の解消）
- スプリッターのドラッグによるサイズ比率変更
- アクティブペインの視覚的強調と切替（クリック）

### 作らないもの

- レイアウトのプリセット保存・呼び出し
- ペインの最大化トグル（将来対応）

## 制約

- レイアウトは共有ストアの二分木 `LayoutNode` を唯一の情報源とする（親 design.md 参照）
- 初期版の UI 操作で作れるのは実用上 2×2 程度まで（データ構造はそれ以上も表現可）

## 完了条件

- 分割操作で 2 ペイン以上にでき、各ペインが独立にタブ・一覧を持つ
- スプリッターをドラッグして比率を変えられる
- ペインを閉じると分割が解消され、残ったペインが領域を埋める

## 画面イメージ

```
+--------------------+‖+--------------------+
| ペイン1 (アクティブ) ‖ | ペイン2             |
+====================+‖+====================+
| ペイン3             ‖ | ペイン4             |
+--------------------+‖+--------------------+
  ‖ = 縦スプリッター  == = 横スプリッター
```
