# folder-tree 概要

## 目的・背景

通常のエクスプローラーと同様に、ウィンドウ左側にフォルダ階層のツリーを表示し、
クリックでアクティブペインのタブを移動できるようにする。
ユーザ要望（2026-07-12）「通常のエクスプローラーのように画面左側にフォルダ選択画面を出したりできるようにしてほしい」を担う。

## スコープ

### 作るもの

- ウィンドウ左端のフォルダツリーパネル（全ペイン共有で 1 つ）
- ルートはホーム＋準備完了状態のドライブ一覧
- ノード展開時にサブフォルダを遅延読込（ファイルは表示しない）
- ノードのクリック（Invoke）でアクティブペインのアクティブタブをそのパスへ移動
- パネルの表示/非表示トグル（折りたたむと細いストリップになる）
- 隠し・システムフォルダは file-list と同じく薄灰色（Opacity 0.55）で表示

### 作らないもの

- ツリーとタブ側の自動同期（タブ移動時にツリーを自動展開・選択）— 将来対応
- ツリー上での右クリックメニュー・リネーム・ドラッグ&ドロップ
- クイックアクセス/ライブラリ/ネットワークノード

## 制約

- AOT 互換を維持（`{x:Bind}` のみ、フィールドベース `[ObservableProperty]` 禁止）
- 遅延読込必須: 起動時にツリー全走査しない（起動時間・メモリを増やさない）
- サブフォルダ列挙は UI スレッドで I/O しない（file-list と同じ非同期パターン）

## 完了条件

- 起動時に左側へツリーが表示され、ドライブ/ホーム配下を展開・移動できる
- アクセス不可フォルダの展開でクラッシュせず、空展開になる
- トグルで折りたたみ/展開ができる
- メモリ増分が軽微（起動時 +数 MB 以内）

## 画面イメージ

```
+--------------+-------------------------------+
| フォルダー  ◀ | [タブバー]                     |
|  ホーム       | [ツールバー]                   |
|   > Desktop  | [ファイル一覧]                 |
|  C:\         |                               |
|   > Users    |                               |
|  D:\         |                               |
+--------------+-------------------------------+
  ◀ = 折りたたみトグル（折りたたみ時は ▶ の細いストリップのみ）
```
