## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 初期表示 | 起動直後 | 左側パネルを確認 | 「フォルダー」ヘッダの下に「ホーム」と準備完了ドライブ（C:\ 等）が表示される | [ ] |
| E-02 | 遅延展開 | E-01 | ホームのシェブロンをクリック | サブフォルダのみが名前昇順で展開される（ファイルは出ない） | [ ] |
| E-03 | クリック移動 | 複数ペイン表示中 | ツリーの任意フォルダ名をクリック | アクティブペインのアクティブタブだけがそのフォルダへ移動し、「戻る」で戻れる | [ ] |
| E-04 | アクセス不可フォルダ | C:\ 直下等 | 「System Volume Information」等を展開 | クラッシュせず子なし（シェブロンが消える）になる | [ ] |
| E-05 | 薄灰色表示 | ホーム直下等 | 隠しフォルダ（AppData 等）のノードを確認 | 一覧と同様に薄灰色（減光）で表示される | [ ] |
| E-06 | 折りたたみ | E-01 | ヘッダの「◀」→「▶」をクリック | パネルが細いストリップに折りたたまれ一覧が広がる。「▶」で元に戻る | [ ] |
| E-07 | 深い階層の展開 | E-02 | 3〜4 階層続けて展開 | 各階層が正しく遅延読込され、UI が固まらない | [ ] |
| E-08 | メモリ増分 | 起動直後 | ツリー表示ありで起動時メモリを比較 | 従来比 +数 MB 以内 | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
