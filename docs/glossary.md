# 用語集

このプロジェクトで使用するドメイン固有の用語を定義する。
実装・設計・仕様書内で用語を使う際はここに定義されたものに統一する。

---

| 用語 | 定義 |
|------|------|
| ペイン | 画面分割された 1 区画。各ペインは独立したタブバー・アドレスバー・ファイル一覧を持つ |
| タブ | ペイン内で開かれた 1 つのフォルダビュー。1 ペインに複数タブを持てる |
| アクティブタブ | ペイン内で現在表示されているタブ |
| アクティブペイン | 最後に操作したペイン。キーボード操作などの対象になる |
| アドレスバー | ペイン上部に表示される、アクティブタブのフォルダパス表示・入力欄 |
| タブ複製 | 「＋」ボタンにより、アクティブタブと同じパスで新しいタブを開く操作 |
| セッション | ペイン構成・各ペインのタブ一覧・各タブのパスなど、アプリの画面状態全体 |
| セッション復元 | アプリ再起動時に、前回終了時のセッションを再現すること |
| ファイル一覧 | タブに表示されるフォルダ内容（名前・更新日時・種類・サイズ）のテーブル |
