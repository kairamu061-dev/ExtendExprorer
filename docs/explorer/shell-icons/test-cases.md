## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 種類別アイコン | .txt/.pdf/.zip 等があるフォルダ | 一覧を表示 | 各拡張子がエクスプローラーと同じ関連付けアイコンで表示される | [x] |
| E-02 | フォルダアイコン | 任意フォルダ | 一覧を表示 | フォルダ行が黄色いフォルダアイコンになる | [x] |
| E-03 | exe 個別アイコン | .exe が複数あるフォルダ | 一覧を表示 | 各 exe が自身の埋め込みアイコンで表示される（共通アイコンにならない） | [x] |
| E-04 | フォールバック | 関連付けのない拡張子 | 一覧を表示 | 白紙アイコンまたは従来グリフで表示（空白・クラッシュしない） | [x] |
| E-05 | スクロール性能 | System32 等の大量フォルダ | 高速にスクロール | 引っかかりなくスクロールでき、表示された行から順にアイコンが付く | [x] |
| E-06 | メモリ増分 | 起動→System32 表示 | 表示後のメモリを従来と比較 | 増分が +数 MB 以内 | [x] |
| E-07 | AOT 安定性 | 正式(AOT)ビルド | E-01〜E-06 を AOT 版で実施 | クラッシュ・WER イベントなし | [x] |

> 検証: 2026-07-19（`動作確認フィードバック_BUG-003とshell-icons_既定アプリ起動_2026-07-19.md` / commit `e63cef4` / run `29258584864` / 正式 AOT ビルド）。E-01〜E-07 全合格。
> E-05: System32（4,500 超ファイル）全域スクロールで平均 48ms/最大 90ms・ステップ、フリーズなし。E-06: アイコン全解決後の増分 +3.7MB。
