# address-bar 概要

## 目的・背景

各ペインで「今どこを見ているか」を常時示し、パス直接入力での移動を可能にする。
ユーザ要求「各画面にフォルダパスが表示される」を担う。

## スコープ

### 作るもの

- ペイン上部のアドレスバー（アクティブタブのフルパス表示）
- パスの直接入力＋Enter による移動、Esc で編集キャンセル
- 不正パス入力時のエラー表示

### 作らないもの

- パンくず（セグメントクリック）形式の表示（将来対応）
- パス補完・履歴サジェスト

## 制約

- 表示・移動対象は常にそのペインのアクティブタブ（親 design.md のストア購読で追従）

## 完了条件

- タブ切替・フォルダ移動のたびにアドレスバーが正しいパスを表示する
- 有効なパスを入力して Enter するとそのフォルダに移動する
- 無効なパスではエラーが表示され、表示パスは元に戻る

## 画面イメージ

```
+--------------------------------------------------+
| パス: [ C:\Users\me\Documents                 ]  |
+--------------------------------------------------+
```
