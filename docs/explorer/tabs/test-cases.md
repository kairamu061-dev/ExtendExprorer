## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 「＋」でタブ複製 | 起動直後（ホーム表示） | 「＋」ボタンをクリック | 同じパスの新タブがアクティブタブの右隣に開き、アクティブになる | [x] |
| E-02 | 複製タブの独立性 | E-01 | 新タブで別フォルダへ移動し、元のタブに切り替える | 元のタブの表示・パスは変わっていない。新タブで「戻る」は無効（履歴は引き継がない） | [x] |
| E-03 | タブ切替 | 複数タブでそれぞれ別フォルダを表示 | タブを順にクリック | 一覧・パス表示・戻る/進むの状態がタブごとに切り替わる | [x] |
| E-04 | タブクローズ | 3 タブ以上で中央のタブをアクティブに | アクティブタブの「×」をクリック | タブが閉じ、右隣のタブがアクティブになる（右端を閉じたら左隣） | [x] |
| E-05 | 最終タブのクローズ | タブが 1 つ | 「×」をクリック | ホームフォルダの新しいタブが開き直される（アプリは終了しない） | [x] |
| E-06 | タブタイトル | 任意のタブ | フォルダを移動する | タブ見出しが移動先のフォルダ名になる（ドライブルートは `C:\` 等） | [x] |
| E-07 | ホイールクリッククローズ | 複数タブ | 非アクティブタブを中クリック | そのタブが閉じる | [x] |
| E-08 | 複製時のソート非引継ぎ | ソートを既定以外に変更 | 「＋」で複製 | 新タブは名前昇順（既定）で表示される | [x] |

E-01〜E-08 は 2026-07-05 の動作確認（run 28737317441, commit `7a24ce9`）で合格。

## ステータス凡例

- `[x]` 合格
- `[!]` 不合格
