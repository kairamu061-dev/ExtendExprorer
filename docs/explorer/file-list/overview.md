# file-list 概要

## 目的・背景

タブに表示するフォルダ内容の一覧は、エクスプローラーの最小単位であり全サブ項目の土台となる。
まず単一ペイン・単一タブで動作する一覧＋ナビゲーションを完成させる。

## スコープ

### 作るもの

- フォルダ内容のテーブル表示（名前・更新日時・種類・サイズ）
- フォルダ→ファイル順の表示と、列ヘッダクリックによるソート
- ダブルクリックによるフォルダ移動、「上へ」「戻る」「進む」ボタン
- 読み込みエラー（アクセス拒否・不存在）の表示

### 作らないもの

- ファイル操作（コピー/削除等）、ファイルの関連付け起動
- アイコン表示・サムネイル（初期版はファイル種類のテキストのみ）

## 制約

- fs アクセスは IPC `fs:list` 経由のみ（親 design.md 参照）

## 完了条件

- 任意のフォルダパスを与えると内容が表示され、ダブルクリック・上へ・戻る/進むで移動できる
- アクセス不能フォルダでエラーメッセージが表示され、アプリが落ちない

## 画面イメージ

```
+--------------------------------------------+
| [↑上へ] [←戻る] [→進む]                    |
|--------------------------------------------|
| 名前 ▲     | 更新日時         | 種類 | サイズ |
| 📁 dir1    | 2026/07/01 12:00 | フォルダ | — |
| file.txt  | 2026/07/02 09:30 | TXT  | 3 KB |
+--------------------------------------------+
```
