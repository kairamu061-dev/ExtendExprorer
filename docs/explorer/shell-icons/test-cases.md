## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 種類別アイコン | .txt/.pdf/.zip 等があるフォルダ | 一覧を表示 | 各拡張子がエクスプローラーと同じ関連付けアイコンで表示される | [ ] |
| E-02 | フォルダアイコン | 任意フォルダ | 一覧を表示 | フォルダ行が黄色いフォルダアイコンになる | [ ] |
| E-03 | exe 個別アイコン | .exe が複数あるフォルダ | 一覧を表示 | 各 exe が自身の埋め込みアイコンで表示される（共通アイコンにならない） | [ ] |
| E-04 | フォールバック | 関連付けのない拡張子 | 一覧を表示 | 白紙アイコンまたは従来グリフで表示（空白・クラッシュしない） | [ ] |
| E-05 | スクロール性能 | System32 等の大量フォルダ | 高速にスクロール | 引っかかりなくスクロールでき、表示された行から順にアイコンが付く | [ ] |
| E-06 | メモリ増分 | 起動→System32 表示 | 表示後のメモリを従来と比較 | 増分が +数 MB 以内 | [ ] |
| E-07 | AOT 安定性 | 正式(AOT)ビルド | E-01〜E-06 を AOT 版で実施 | クラッシュ・WER イベントなし | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
