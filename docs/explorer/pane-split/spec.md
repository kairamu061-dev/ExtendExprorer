# pane-split 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 縦/横分割 | ツールバーの「縦分割」「横分割」でアクティブペインを 2 分割。新ペインはアクティブタブと同じパスのタブを 1 つ持ち、アクティブペインになる |
| 2 | ペインクローズ | ペインの最後のタブが閉じられたとき分割を解消し、兄弟ペインが領域を埋める |
| 3 | サイズ変更 | ペイン境界のスプリッターをドラッグして比率（ratio）を変更。最小 10% |
| 4 | アクティブペイン切替 | ペイン内の任意箇所クリックでそのペインをアクティブ化。アクティブペインは枠線で強調 |

## 画面・操作フロー

1. 「縦分割」/「横分割」をクリック → アクティブペインが SplitNode に置換され、右/下に新ペイン
2. スプリッターをドラッグ → ratio 更新、両側のペインがリサイズ
3. ペインの最後のタブを「×」→ ペイン削除、親 SplitNode が兄弟ノードに置換される

## 各画面 / 状態の詳細

### 分割レイアウト

- **表示内容**: LayoutNode 二分木を再帰的に flex レイアウトへ描画。SplitNode → flex コンテナ＋スプリッター、PaneLeaf → ペイン DOM
- **操作**: スプリッタードラッグ（カーソルは col-resize / row-resize）
- **遷移先**: なし

### アクティブペイン

- **表示内容**: 枠線 2px #0078D4。非アクティブは 1px #D0D0D0
- **操作**: クリックで切替
- **遷移先**: なし

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| ペイン数が上限（8）で分割 | 何もしない（ボタン無効化） |
| ドラッグで最小比率を下回る | 10% で停止 |

## 未対応ケース

- ペインの入れ替え・ドラッグ移動
- 分割方向の後からの変更（一度閉じて分割し直す）
