## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | Ctrl+C→Ctrl+V | ファイル選択 | 同フォルダで実行 | 「〜 - コピー」が生成される | [ ] |
| E-02 | Ctrl+X→Ctrl+V | ファイル選択 | 別フォルダ表示後に Ctrl+V | 移動される（元から消える） | [ ] |
| E-03 | 複数選択のコピー | Ctrl/Shift で複数選択 | Ctrl+C → 別フォルダへ Ctrl+V | 選択した全項目がコピーされる | [ ] |
| E-04 | エクスプローラー相互運用 | エクスプローラーで Ctrl+C | 本アプリで Ctrl+V（およびその逆） | 双方向で貼り付けできる | [ ] |
| E-05 | Delete | ファイル選択 | Delete | ごみ箱へ移動（シェルの動作） | [ ] |
| E-06 | 空操作 | 選択なし / クリップボード空 | 各ショートカット | 何も起きずクラッシュしない | [ ] |
| E-07 | サイドボタン | 履歴のあるタブ | XButton1 / XButton2 | 戻る/進む。履歴端では無反応 | [ ] |
| E-08 | AOT 安定性 | 正式(AOT)ビルド | E-01〜E-07 | クラッシュ・WER なし | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
