# explorer 概要

## 目的・背景

Windows 11 標準エクスプローラーでは複数フォルダの同時閲覧（画面分割）と、終了時のタブ・レイアウト状態の完全な復元ができない。
本機能エリアは ExtendExprorer の中核であるマルチタブ・マルチペインのファイル閲覧 UI を提供する。

## スコープ

### 作るもの

- ペインの画面分割（分割の追加・削除、スプリッターによるサイズ調整）
- ペインごとのタブバー（タブの追加・複製・切替・クローズ・「＋」ボタン）
- ペインごとのアドレスバー（アクティブタブのフォルダパス表示・パス入力による移動）
- ファイル一覧表示（名前・更新日時・種類・サイズ、ダブルクリックでフォルダ移動）
- セッション保存・復元（終了時の状態を JSON に保存し、次回起動時に再現）

### 作らないもの

- ファイル操作（コピー/移動/削除/リネーム）
- 検索・フィルタ機能
- コンテキストメニュー（右クリック）からの Windows シェル連携

## 制約

- Electron（Node.js main + HTML/CSS/TS renderer）で実装する
- Renderer から直接 fs にアクセスしない（contextBridge 経由の IPC のみ）
- 開発・動作確認は Linux devcontainer 上、実行ターゲットは Windows 11

## 完了条件

- 複数タブを開き、タブ間を切り替えられる
- 画面を 2 分割以上にでき、各ペインが独立にフォルダを表示できる
- 各ペインにアクティブタブのフォルダパスが表示される
- アプリを閉じて再起動すると、ペイン構成・タブ・各タブのパスが復元される
- 「＋」ボタンでアクティブタブと同じパスのタブが複製される
