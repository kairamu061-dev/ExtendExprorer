# file-list 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 一覧表示 | 指定パスの内容を名前・更新日時・種類・サイズで表示。フォルダ先頭、次にファイル |
| 2 | ソート | 列ヘッダクリックで昇順/降順を切替（既定: 名前昇順。フォルダ先頭は維持） |
| 3 | フォルダ移動 | フォルダ行のダブルクリックでそのフォルダへ移動 |
| 4 | 履歴移動 | 「戻る」「進む」でタブの履歴を移動、「上へ」で親フォルダへ |
| 5 | エラー表示 | 列挙失敗時、一覧領域にエラーメッセージを表示 |

## 画面・操作フロー

1. タブのパスが変わる（初期表示・移動・履歴移動）→ `fs:list` で列挙し一覧を再描画
2. フォルダ行をダブルクリック → タブの履歴に追加してパス変更 → 1 に戻る
3. 「上へ」→ 親パスへ移動（ルートでは無効化）
4. 「戻る」「進む」→ `historyIndex` を移動して再描画（端では無効化）

## 各画面 / 状態の詳細

### 一覧テーブル

- **表示内容**: 種類は拡張子の大文字表記（フォルダは「フォルダ」）。サイズはフォルダは「—」、ファイルは KB/MB 表記。更新日時は `YYYY/MM/DD HH:mm`
- **操作**: 行クリックで選択（ハイライトのみ）、ヘッダクリックでソート
- **遷移先**: フォルダのダブルクリックで同一タブ内パス変更

### エラー状態

- **表示内容**: 一覧領域中央に「アクセスが拒否されました」「パスが見つかりません」等
- **操作**: 「上へ」「戻る」は引き続き使用可能
- **遷移先**: 移動操作で正常状態に復帰

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| EACCES / EPERM | 「アクセスが拒否されました」 |
| ENOENT | 「パスが見つかりません」 |
| その他の fs エラー | 「読み込みに失敗しました: {メッセージ}」 |

## 未対応ケース

- ファイルダブルクリックでの外部アプリ起動（将来対応）
- 隠しファイルの表示切替（初期版は常に表示）
- 大量ファイル（1 万件超）の仮想スクロール（性能問題が出たら対応）
