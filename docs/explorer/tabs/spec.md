# tabs 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | タブ表示 | ペイン上部にタブを開いた順に表示。タイトルはフォルダ名（ルートはパスそのもの） |
| 2 | タブ切替 | タブクリックでアクティブタブを変更し、一覧・アドレスバーを更新 |
| 3 | タブ複製 | 「＋」ボタンでアクティブタブと同じパスの新規タブをアクティブタブの右隣に追加しアクティブ化。履歴は引き継がない |
| 4 | タブクローズ | タブの「×」で閉じる。アクティブタブを閉じたら右隣（なければ左隣）をアクティブ化 |
| 5 | 最終タブの扱い | ペインの最後のタブを閉じたらペインを閉じる。最後のペインの場合はホームのタブを開き直す |

## 画面・操作フロー

1. タブをクリック → `activeTabId` 変更 → ファイル一覧・アドレスバーが再描画
2. 「＋」をクリック → 新規 Tab（path = アクティブタブの path）を挿入しアクティブ化
3. 「×」をクリック → タブ削除 → 仕様 4 / 5 に従いアクティブタブ・ペインを調整

## 各画面 / 状態の詳細

### タブバー

- **表示内容**: 各タブ（タイトル＋「×」）と末尾の「＋」ボタン。アクティブタブは背景色 #E5F1FB・下線 #0078D4 で強調。タブ幅は最大 160px、超過分は省略記号
- **操作**: クリック（切替）、×（クローズ）、＋（複製）。ホイールクリックでもクローズ
- **遷移先**: なし（ペイン内で完結）

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| タブ数が上限（50）に達した状態で「＋」 | 何もしない（ボタン無効化） |

## 未対応ケース

- タブの並べ替え（ドラッグ）・ペイン間移動
- Ctrl+T / Ctrl+W 等のキーボードショートカット（将来対応）
