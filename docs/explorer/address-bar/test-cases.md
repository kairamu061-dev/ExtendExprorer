## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | パンくず表示 | 任意フォルダ表示 | アドレスバーを見る | `[C:] › [Users] › …` のようにフォルダ単位で表示される | [ ] |
| E-02 | セグメント移動 | 深い階層を表示 | 途中のフォルダ名をクリック | そのフォルダへ移動し、パンくずが更新される（戻るで復帰可） | [ ] |
| E-03 | 表示追従 | 複数タブ/移動 | タブ切替・フォルダ移動・戻る/進む | 常にアクティブタブの現在パスを表示 | [ ] |
| E-04 | 編集モード | 任意フォルダ表示 | パンくずの余白（フォルダ名以外）をクリック | フルパスが全選択された編集ボックスになる | [ ] |
| E-05 | パス入力移動 | E-04 | 有効なパスを入力して Enter | そのフォルダへ移動し、パンくず表示に戻る | [ ] |
| E-06 | ファイルパス入力 | E-04 | ファイルのフルパスを入力して Enter | その親フォルダへ移動する | [ ] |
| E-07 | 無効パス | E-04 | 存在しないパスを入力して Enter | 「パスが見つかりません」が 3 秒表示され、移動しない（編集継続） | [ ] |
| E-08 | Esc 取消 | E-04 | Esc | 編集を破棄しパンくず表示に戻る（移動しない） | [ ] |
| E-09 | フォーカス喪失で復帰 | E-04 | Enter せず他所をクリック | 編集を破棄し元のパンくず表示に戻る | [ ] |
| E-10 | AOT 安定性 | 正式(AOT)ビルド | E-01〜E-09 | クラッシュ・WER なし | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
