## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | ペイン間移動 | 2 ペイン表示 | 左のファイルを右の一覧へドラッグ | 移動される（元から消える） | [~] |
| E-02 | Ctrl でコピー | 同上 | Ctrl を押しながらドロップ | コピーされる（元が残る） | [~] |
| E-03 | 複数項目 | 複数選択してドラッグ | 別ペインへドロップ | 全項目が移動される | [~] |
| E-04 | フォルダの D&D | フォルダをドラッグ | 別ペインへドロップ | フォルダごと移動される | [~] |
| E-05 | 本アプリ→エクスプローラー | 両方表示 | 本アプリからエクスプローラーへドロップ | エクスプローラー側に移動/コピーされる | [~] |
| E-06 | エクスプローラー→本アプリ | 同上 | エクスプローラーから本アプリへドロップ | 表示中フォルダへ移動/コピーされる | [~] |
| E-07 | 同フォルダへのドロップ | 同一ペイン内 | 同じ一覧へドロップ | 何も起きない | [~] |
| E-08 | AOT 安定性 | 正式(AOT)ビルド | E-01〜E-07 | クラッシュ・WER なし | [~] |

> 検証: 2026-07-20。合成マウス（SetCursorPos+mouse_event）の連続ドラッグを WinUI の D&D が受け付けず**自動化不可**。ドラッグ試行中もクラッシュなし（WER 0）。E-01〜E-08 は実機目視での確認が必要（pane-split E-05 と同じ制約）。

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
