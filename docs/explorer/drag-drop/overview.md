# drag-drop 概要

## 目的・背景

ファイル・フォルダをドラッグ&ドロップで移動/コピーできるようにする。
ユーザ要望（2026-07-19）「ファイル・フォルダをドラッグアンドドロップで移動できるようにしたい」を担う。
ペイン間だけでなく、エクスプローラー等の外部アプリとの間でも D&D できることを目標にする。

## スコープ

### 作るもの

- 一覧の項目（複数選択可）をドラッグ元にする（StorageItems 形式 = エクスプローラー互換）
- 一覧をドロップ先にする: 表示中フォルダへ移動（Ctrl 押下でコピー）
- ペイン間 D&D / エクスプローラー⇔本アプリの D&D
- 実ファイル操作は IFileOperation（進捗・衝突ダイアログはシェル任せ）

### 作らないもの

- 一覧内のサブフォルダ行への直接ドロップ（表示中フォルダのみが対象。将来対応）
- folder-tree ノードへのドロップ（将来対応）
- 同一ボリューム=移動/別ボリューム=コピーの自動判別（既定は常に移動、Ctrl でコピー）

## 制約

- WinUI 標準の D&D API（CanDragItems / AllowDrop / DataPackage）のみ使用（OLE DoDragDrop を自前実装しない）
- AOT 互換

## 完了条件

- ペイン間でファイルをドラッグして移動/コピーできる
- エクスプローラーとの間で双方向に D&D できる
- 同フォルダへのドロップは何も起きない（誤操作で複製されない）
