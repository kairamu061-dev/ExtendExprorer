# tabs 概要

## 目的・背景

1 ペイン内で複数フォルダを切り替えながら閲覧するためのタブ機能。
ユーザ要求「タブを複数開ける」「＋ボタンでタブの複製ができる」を担う。

## スコープ

### 作るもの

- ペインごとのタブバー（タブの表示・切替・クローズ）
- 「＋」ボタンによるアクティブタブの複製（同じパスで新規タブ）
- タブタイトル（フォルダ名）の表示と更新

### 作らないもの

- タブのドラッグ並べ替え・ペイン間移動（将来対応）
- タブのピン留め・色分け

## 制約

- タブ状態は共有ストアの `PaneLeaf.tabs` を唯一の情報源とする（親 design.md 参照）

## 完了条件

- 「＋」でアクティブタブと同じパスのタブが右隣に開き、アクティブになる
- タブクリックで表示が切り替わり、「×」でタブが閉じる
- 最後のタブを閉じたときの挙動が仕様どおり（ペインを閉じる / 最後のペインならホームを開き直す）

## 画面イメージ

```
+----------------------------------------------+
| [ Documents ×][ photos ×][ work ×] [+]        |
+----------------------------------------------+
   ^アクティブタブは背景色で強調
```
