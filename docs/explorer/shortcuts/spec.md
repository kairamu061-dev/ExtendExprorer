# shortcuts 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | Ctrl+C / Ctrl+X | 選択中の項目（複数可）を CF_HDROP＋Preferred DropEffect(コピー=5/移動=2)でクリップボードへ |
| 2 | Ctrl+V | クリップボードの内容を表示中フォルダへ貼り付け（背景メニューの「貼り付け」と同一処理） |
| 3 | Delete | 選択中の項目をごみ箱へ（シェルの確認・進捗ダイアログ付き） |
| 4 | サイドボタン | XButton1=アクティブタブの「戻る」、XButton2=「進む」。ペイン内クリック位置を問わない |

## 画面・操作フロー

1. 一覧で項目を選択（Shift/Ctrl の複数選択可） → Ctrl+C / Ctrl+X
2. 任意のフォルダを表示 → Ctrl+V → シェルがコピー/移動を実行（同フォルダコピーは「- コピー」自動リネーム）
3. Delete → ごみ箱へ移動

## 各画面 / 状態の詳細

（既存画面への入力追加のみ。表示の変更なし）

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 選択なしで Ctrl+C/X/Delete | 何もしない |
| クリップボードにファイルが無い状態で Ctrl+V | 何もしない |
| 貼り付け・削除の失敗 | シェルのダイアログに委ねる（アプリは落ちない） |
| 履歴端でサイドボタン | 何もしない（ボタン無効時と同じ） |

## 未対応ケース

- 切り取り中の項目の半透明表示（エクスプローラーの視覚効果）
- Ctrl+Z（元に戻す）— FOF_ALLOWUNDO は付けているためエクスプローラー側からは Undo 可能
