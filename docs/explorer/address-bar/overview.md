# address-bar 概要

## 目的・背景

各ペインで「今どこを見ているか」を常時示し、パス直接入力での移動を可能にする。
ユーザ要求「各画面にフォルダパスが表示される」を担う。

## スコープ

### 作るもの

- ペイン上部のアドレスバー（アクティブタブのパスをパンくず表示）
- 各フォルダ名（パンくずセグメント）クリックでそのフォルダへ移動
- 余白クリックでフルパス全選択の編集モードへ。Enter で移動、Esc/フォーカス喪失で元表示に復帰
- 不正パス入力時のエラー表示（3 秒「パスが見つかりません」）

### 作らないもの

- パス補完・履歴サジェスト
- 環境変数（%USERPROFILE% 等）の展開

## 制約

- 表示・移動対象は常にそのペインのアクティブタブ（親 design.md のストア購読で追従）

## 完了条件

- タブ切替・フォルダ移動のたびにアドレスバーが正しいパスを表示する
- パンくずの各フォルダ名クリックでそのフォルダへ移動できる
- 余白クリックで全選択編集に入り、有効なパスを Enter で移動できる
- 無効なパスではエラーが表示され、Enter 以外でフォーカスが外れると元のパス表示に戻る

## 画面イメージ

表示（パンくず）:

```
+--------------------------------------------------+
| [C:] › [Users] › [me] › [Documents]              |
+--------------------------------------------------+
```

編集（余白クリック時・全選択）:

```
+--------------------------------------------------+
| [ C:\Users\me\Documents                      ]  |
+--------------------------------------------------+
```
