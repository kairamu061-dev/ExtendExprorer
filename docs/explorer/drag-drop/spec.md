# drag-drop 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ドラッグ元 | 選択中の項目（複数可）を StorageItems としてドラッグ開始。外部アプリへのドロップ可 |
| 2 | ドロップ先 | 一覧上へのドロップで表示中フォルダへ**移動**。Ctrl 押下中は**コピー**（カーソル表示も切替） |
| 3 | ペイン間 D&D | 左ペインの項目を右ペインの一覧へドロップ等。移動/コピーはシェルが実行 |
| 4 | 外部連携 | エクスプローラー→本アプリ、本アプリ→エクスプローラーの双方向 |

## 画面・操作フロー

1. 項目を選択しドラッグ開始 → ドラッグビジュアルが表示される
2. 別ペインの一覧（または外部アプリ）へドロップ → シェルの進捗ダイアログでファイル操作が実行される
3. Ctrl を押しながらドロップ → コピーになる

## 各画面 / 状態の詳細

（既存の一覧のみ。ドロップ受付中はカーソルに移動/コピーのバッジが付く＝WinUI 標準）

## エラーケース

| 条件 | メッセージ / 挙動 |
|------|------------------|
| 同じフォルダへの移動ドロップ | 何もしない |
| ファイル以外（テキスト等）のドロップ | 受け付けない（None 表示） |
| ドロップ処理の失敗 | シェルのダイアログに委ねる（アプリは落ちない） |
| ドラッグ中に元ファイルが消えた | 消えた項目はスキップ |

## 未対応ケース

- サブフォルダ行・ツリーノードへのドロップ
- 同一ボリューム判定による移動/コピーの自動切替（Shift での強制移動）
