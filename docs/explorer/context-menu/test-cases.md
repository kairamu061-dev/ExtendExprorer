## E2Eテスト（手動）

| ID | テスト名 | 前提条件 | 手順 | 期待結果 | ステータス |
|----|---------|---------|------|---------|-----------|
| E-01 | 項目メニュー表示 | 任意フォルダ表示中 | ファイルを右クリック | 項目が選択され、カーソル位置にエクスプローラー同等のメニューが出る | [ ] |
| E-02 | 開く | E-01 | テキストファイルで「開く」等を実行 | 既定アプリで開く | [ ] |
| E-03 | コピー→貼り付け | E-01 | ファイルで「コピー」→空白右クリック「貼り付け」 | 同フォルダに「- コピー」が作られる（表示は再移動で確認） | [ ] |
| E-04 | 削除 | E-01 | 不要ファイルで「削除」 | シェルの確認/ごみ箱動作がそのまま行われる | [ ] |
| E-05 | プロパティ | E-01 | 「プロパティ」を実行 | シェルのプロパティダイアログが開く | [ ] |
| E-06 | 送る/プログラムから開く | E-01 | 「送る」「プログラムから開く」をポイント | サブメニューの中身が空でなく展開される（IContextMenu2/3 転送の確認） | [ ] |
| E-07 | 背景メニュー | 任意フォルダ表示中 | 一覧の空白部分を右クリック | 「新規作成」「貼り付け」等を含む背景メニューが出る | [ ] |
| E-08 | 新規作成 | E-07 | 「新規作成」→「フォルダー」 | フォルダが作られる（表示は再移動で確認） | [ ] |
| E-09 | キャンセル | E-01 | メニュー外クリック / Esc | 何も実行されず閉じる | [ ] |
| E-10 | フォルダ項目のメニュー | 任意フォルダ表示中 | サブフォルダを右クリック | フォルダ向けメニュー（開く/プロパティ等）が出て動作する | [ ] |
| E-11 | AOT 安定性 | 正式(AOT)ビルド | E-01〜E-10 を AOT 版で実施 | クラッシュ・WER イベントなし | [ ] |

## ステータス凡例

- `[ ]` 未実施
- `[~]` 実施中
- `[x]` 合格
- `[!]` 不合格
